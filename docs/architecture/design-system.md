# Design System

Tema escuro obrigatório — tokens Tailwind slate/emerald.

| Elemento | Classe |
| --- | --- |
| Page wrapper | `bg-slate-950 text-slate-100` |
| Card / painel | `bg-slate-900 border border-slate-800 rounded-xl` |
| Input | `bg-slate-800 border border-slate-700 ... focus:ring-emerald-500` |
| Botão primário | `bg-emerald-500 hover:bg-emerald-400 text-white` |
| Botão ghost | `bg-emerald-500/10 hover:bg-emerald-500/20 text-emerald-400` |
| Badge ativo | `bg-emerald-500/10 text-emerald-400 rounded-full` |
| Badge neutro | `bg-slate-800 text-slate-400 rounded-full` |

**Proibido:** `bg-white`, `bg-gray-*`, `text-gray-900`, `ring-blue-500`

## i18n

Idiomas obrigatórios: `pt`, `en`, `fr`, `es` — nunca hardcodar strings visíveis no JSX.
